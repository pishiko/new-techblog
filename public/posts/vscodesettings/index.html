VSCodeの見た目を自分好みにする

2021-02-27

VSCode の設定


変えすぎて何が何だかよくわからなくなってきそうなので整理も兼ねて設定を並べていきたいと思います．


テーマ

Solarized Dark を使用しています．
Solarized

"workbench.colorTheme": "Solarized Dark",


フォント

半角が Cancadia Code，全角日本語が Han Code になるように設定しています． Cancadia Code は Microsoft の作った WindowsTerminal のデフォルトフォントで，リガチャにも対応しています． 日本語は見やすさ重視です．
microsoft/cascadia-code: This is a fun, new monospaced font that includes programming ligatures and is designed to enhance the modern look and feel of the Windows Terminal.
adobe-fonts/source-han-code-jp: Source Han Code JP | 源ノ角ゴシック Code

"editor.fontFamily": "Cascadia Code, Han Code JP",

リガチャ（合字）は個別に有効にする必要があります．

"editor.fontLigatures": true,


ターミナルの配色
